# streamglass

Streaming layers and utilities for Twitch/OBS

## Getting Started

Install the Vercel CLI

```bash
npm i -g vercel
```

```bash
npm install --workspaces
```

### Frontend

```bash
 npm run dev --workspace=frontend
```

### Server

```bash
 npm run start --workspace=server
```

## Testing

```bash
npm run test --workspaces
```
